La propiedad text-decoration

La propiedad text-decoration especifica cómo el texto será decorado.

Los valores usados comúnmente son:
none
- El valor por defecto, este define un texto como normal
inherit
- Hereda esta propiedad de su elemento padre
overline
- Dibuja una línea horizontal encima del texto
underline
- Dibuja una línea horizontal debajo del texto
line-through
- Dibuja una línea horizontal atravesando el texto (substituye la etiqueta HTML s)

Puedes combinar los valores underline, overline, y line-through en una lista separada por espacios para añadir múltiples líneas decorativas.

La propiedad text-decoration

NOTA
