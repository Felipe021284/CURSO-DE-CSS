Los valores de la propiedad cursor

Hay muchos otros posibles valores para la propiedad cursor, tales como:
default -
el cursor por defecto
crosshair -
el cursor se muestra como una cruz o suma
pointer -
el cursor se muestra como un ícono de mano La lista de valores posibles es bastante larga.

El valor predeterminado

Generalmente, el aspecto del cursor del ratón es alterado para proporcionar una experiencia más interesante a los visitantes del sitio Web. Sin embargo, elegir el estilo del cursor incorrecto puede ser también engañoso.

Por ejemplo, si el valor del cursor está establecido como default, los usuarios pueden que no "vean" los enlaces.

nota

Escoge el estilo del cursor del ratón cuidadosamente.
