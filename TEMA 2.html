CSS en línea

Usar un estilo en línea es una de las formas de insertar una hoja estilo. Con un estilo en línea, un único estilo es aplicado a un único elemento.

Para usar un estilo en línea, añade el atributo de estilo a la etiqueta respectiva.

CSS incrustado/interno

Estilos internos son definidos dentro del elemento style , en la sección head de una página web.

NOTA
Puede utilizarse una hoja de estilo interno si una página única tiene un estilo único.
