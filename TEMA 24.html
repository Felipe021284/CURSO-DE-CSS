Los tamaños máximos y mínimos

Para establecer la altura y el ancho mínimo y máximo de un elemento, puedes usar las siguientes propiedades:
min-width - el ancho mínimo de un elemento
min-height - la altura mínima de un elemento
max-width - el ancho máximo de un elemento
max-height - la altura máxima de un elemento
