Estableciendo estilos a los enlaces

Se les puede dar estilo a los enlaces con cualquier propiedad CSS (por ejemplo, color, font-family, background, Etc.).
Adicionalmente, se les puede dar diferentes estilos a los enlaces, dependiendo de en qué estado se encuentren. Los siguientes pseudo selectores están disponibles:
a:link -
define el estilo para enlaces normales no visitados
a:visited -
define el estilo para enlaces visitados
a:active -
un enlace se convierte en activo una vez que se hace clic en él
a:hover -
ocurre cuando el ratón está sobre el enlace

NOTA

Al establecer el estilo de varios estado de los enlaces, hay algunas reglas acerca del orden:
- a:hover DEBE venir después de a:link y a:visited
- a:active DEBE venir después de a:hover
