La propiedad overflow

Como comentamos anteriormente, cada elemento de la página es una caja. Si no se establece la altura de la caja, la altura de esa caja crecerá tanto como sea necesario para acomodar el contenido.

Este texto está dentro del elemento div, que tiene un azul color de fondo y flota hacia la izquierda. Establecemos un específico alto y ancho para el elemento div, y como puede ver, el contenido no puede caber.












NOTA

La propiedad CSS overflow especifica el comportamiento que ocurre cuando el contenido de un elemento se desborda de la caja del elemento.

Los valores de la propiedad overflow

Hay cuatro valores para la propiedad overflow:
visible (el valor por defecto),
scroll,
hidden,
y auto.

El valor scroll resulta en un desbordamiento recortado, pero una barra de desplazamiento es agregada, por lo que se puede ver el resto del contenido.
