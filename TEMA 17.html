La propiedad text-transform

La propiedad text-transform de CSS especifica como pasar a mayúsculas el texto de un elemento.
Por ejemplo, puede ser usada para hacer que el texto aparezca con la inicial de cada palabra en mayúsculas.

ejemplo del valor capitalize este convierte la primera letra en mayuscula

Valores de text-transform

Utilizando la propiedad
text-transform puedes hacer que el texto aparezca todo en mayúsculas o minúsculas.
