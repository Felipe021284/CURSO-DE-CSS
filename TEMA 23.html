La propiedad border

La propiedad border CSS te permite personalizar los bordes de los elementos HTML.
Con el fin de agregar un borde al elemento, necesitas especificar el tamaño, estilo y color del borde.

NOTA

Ninguna de las propiedades del borde tendrá efecto a menos que la propiedad border-style esté establecida.

La propiedad border-style

El valor por defecto de border-style es none, lo cual define ningún borde. Hay varios estilos admitidos para la propiedad border-style:
dotted (punteado),
dashed (discontinua),
double (doble),
Etc.

nota

En CSS, es posible especificar diferentes bordes para diferentes lados, usando las siguientes propiedades: border-top-style (superior), border-right-style (derecho), border-bottom-style (inferior) y border-left-style (izquierdo) para los lados correspondientes.
