Posicionando elementos

La propiedad CSS positioning te permite posicionar un elemento. También puede posicionar un elemento detrás de otro y especificar qué debe pasar cuando el contenido de un elemento es demasiado grande.

Los elementos pueden ser posicionados usando las propiedades top, bottom, left y right. Sin embargo, estas propiedades no funcionarán a menos que la propiedad position sea establecida primero. Ellos también funcionan diferente dependiendo del método de posicionamiento.

Posicionamiento estático

Los elementos HTML son posicionados estáticos por defecto.
Un elemento posicionado como estático siempre será posicionado de acuerdo al flujo normal de la página.

Posicionamiento fijo

Un elemento con un posicionamiento fijo es posicionado relativo a la ventana del navegador, y no se moverá aunque la ventana sea desplazada. La posición puede ser especificada usando una o más de las propiedades
top, right, bottom y left.

Posicionamiento relativo

Un elemento con posicionado como relativo es posicionado relativo a su posición normal.
Las propiedades
top, right, bottom y left
pueden ser usadas para especificar cómo la caja a dibujar será desplazada.

El contenido de elementos posicionados de forma relativa puede moverse y superponerse sobre otros elementos, pero el espacio reservado para el elemento aún se preserva en el flujo normal.

Este valor no puede ser usado para celdas de tabla, columnas, grupos de columnas, filas, grupos de filas, o leyendas.

Posicionamiento absoluto

Un elemento con posicionamiento absoluto es posicionado relativo al primer elemento padre que tenga una posición distinta a estático. Si no se haya tal elemento, el bloque contenedor es html.

Los elementos con posicionamiento absoluto son removidos del flujo normal. El documento y otros elementos se comportan como si el elemento posicionado de manera absoluta no existiera.

NOTA

Elementos posicionados como estáticos no son afectados por las propiedades top, bottom, left y right.

Elementos posicionados como fijos (fixed) son removidos del flujo normal. El documento y otros elementos se comportan como si el elemento posicionado como fijo no existiera. Los elementos posicionados como fijos pueden superponerse sobre otros elementos.

Elementos con posicionamiento absoluto pueden superponerse sobre otros elementos.
