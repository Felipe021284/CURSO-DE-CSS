display: block

Cada elemento en una página web es una caja rectangular. La propiedad display determina cómo se comporta esa caja rectangular. Un elemento de bloque e un elemento que ocupa el ancho máximo disponible, con saltos de línea antes y después.

JE NOTA

Establecer la propiedad display del elemento sólo cambia cómo se muestra el elemento, no el tipo de elemento que es. Por lo tanto, un elemento en línea con display:block no se le permite tener otros elementos de bloque dentro de él.

display:none

display:none oculta un elemento, por lo que no ocupará ningún espacio. El elemento estará oculto, y la página será desplegada como si el elemento no estuviera allí.
