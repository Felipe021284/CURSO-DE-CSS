El Modelo de Caja de CSS

Todos los elementos HTML pueden ser considerados como cajas.
El Modelo de Caja de CSS representa el diseño y disposición de la página.
Consiste en márgenes, bordes, relleno, y el contenido propiamente.

Las propiedades operan en el mismo orden: superior, derecha, inferior e izquierda.

NOTA

El término "box model" (modelo de caja) se utiliza cuando se habla de diseño.
