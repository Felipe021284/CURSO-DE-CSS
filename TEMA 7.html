FONT SIZE

La propiedad font-size

La propiedad font-size establece el tamaño de una letra. Una forma de establecer el tamaño de las fuentes en la web es utilizando palabras clave.
Por ejemplo
xx-small, small, medium, large, larger, Etc.

NOTA
Las palabras claves son útiles si no quieres que el usuario pueda aumentar el tamaño de la fuente ya que afectaría adversamente la apariencia del sitio.

La propiedad font-size

También puedes utilizar valores numéricos en píxeles o ems para manipular el tamaño de la fuente.
Fijar el tamaño de la fuente en píxeles (px) es una buena opción cuando necesitas precisión a nivel de píxel, dándote control total sobre el tamaño del texto.

La unidad de tamaño em es otra forma de fijar el tamaño de fuente (em es una unidad de tamaño relativa).
Permite que la mayoría de los navegadores redimensione el texto. Si no has establecido el tamaño de fuente en ninguna parte de la página, se utilizará entones el tamaño por defecto del navegador, el cual es 16px.

Para calcular el tamaño de un em, puedes usar la siguiente fórmula: em = pixels / 16
